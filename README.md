# YOLOVs

这个仓库用于集中管理 YOLO 相关项目源码与学习笔记，包含 YOLOv3 / YOLOv5 / YOLOv8（Ultralytics）等内容。

## 目录结构

- `YOLOv3-AAdown/`：YOLOv3 相关代码
- `yolov5/`：YOLOv5 源码与学习总结
- `ultralytics/`：Ultralytics（YOLOv8 等）源码与学习总结
- `smartcar/`：编译与工具脚本
- `compiler_flow.png`：编译流程图

## 学习笔记

- `yolov5/学习总结.md`
- `ultralytics/YOLOv8检测头数学总结.md`

## 使用说明

1. 克隆仓库：

```bash
git clone https://github.com/poison-hip/YOLOVs.git
cd YOLOVs
```

2. 进入对应子目录按项目文档运行：

- `yolov5/README.md`
- `ultralytics/README.md`
- `YOLOv3-AAdown/README.md`

## 说明

- 本仓库已将各子项目作为普通目录纳入统一版本管理（非子模块）。
- 如需同步上游官方仓库，请在对应子目录手动处理。
